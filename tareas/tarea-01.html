Juego Snake

Descripción del proyecto

En esta tarea desarrollé un videojuego clásico tipo "Snake" utilizando HTML, CSS y JavaScript. El objetivo es controlar una serpiente que crece al comer comida sin chocar contra las paredes ni contra sí misma.

Funcionamiento

  • El jugador controla la dirección con las flechas del teclado
  • La serpiente crece al comer alimento
  • El juego termina al colisionar con los bordes o consigo misma
  • El juego se reinicia automáticamente al perder

Tecnologías utilizadas

  • HTML para la estructura
  • CSS para el diseño visual
  • JavaScript para la lógica del juego
